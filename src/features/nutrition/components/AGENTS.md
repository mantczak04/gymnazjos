# Nutrition Components

This directory contains screen-level and feature-specific React components for nutrition.

- Components may compose UI, forms, hooks, and query mutations.
- Keep nutrition calculations in `utils` and database operations in `queries`.
- Keep food logging fast on mobile: direct controls, lightweight dialogs, and minimal navigation.
- Use shared UI primitives from `src/shared/components/ui`.
- Keep all UI text in English and aligned with `.ai/docs/diet.md`.
