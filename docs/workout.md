# Workout MVP

The workout module supports:

- Exercise Library
- Workout Templates
- Starting a workout from an active template
- One active local workout draft
- Add/edit/delete sets
- Add ad-hoc exercises from the Exercise Library
- Last performed sets
- Required 1-5 post-workout feeling rating
- Completed workout history
- Basic saved workout editing
- Weighted volume and reps-only volume metrics

Excluded from the MVP: Supabase Auth, multiple active workouts, empty workouts, RPE, rest timers, drag-and-drop ordering, personal records, workout charts, nutrition, and dashboard functionality.
